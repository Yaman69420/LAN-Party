# Database Exports

Plaats hier je phpMyAdmin exports (`lan_party_db.sql`).

## Workflow

1. Export database in phpMyAdmin (Format: SQL)
2. Zet bestand hier als `lan_party_db.sql`
3. Commit en push
4. Teamgenoot importeert in hun phpMyAdmin

> ⚠️ Let op: bij import worden bestaande tabellen overschreven!
